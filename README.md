### Features
- Upload custom image or use provided examples
- Posterization / custom pallete color quantization
- Convert image to pennies mosaic
- Convert image to Rubik's cube mosaics
- Convert image to BEHR paint swatches mosaic
- Convert image to bottle caps mosaic
- Ability to remove bottlecaps and pennies that aren't available
- Convert image to dice mosaic
- Hover over preview to see exact color/code
